Para fazer este projeto rodar na sua maquina:

Passo 1: Baixar e instalar o NodeJS versão 12.2

Passo 2: Acessar a pasta UI, abrir um prompt de comando e rodar o seguinte comando: npm i

Passo 3: Acessar a pasta backend, abrir um prompt de comando e rodar o seguinte comando: npm i

Após realizar estes passos, todas as dependências do projeto estarão instaladas e prontas para uso.

Para compilar o projeto:

O projeto deve ser compilado em duas partes diferentes, importante compilar em ordem. Backend -> Frontend.

Compilar o back-end: Acessar a pasta backend, abrir um prompt de comando e rodar o comando: npm run dev

Compilar o front-end: Acessar a pasta UI, abrir um prompt de comando e rodar o comando: npm start.

Mantenha os dois prompts abertos, pois eles devem estar rodando em tempo real.

Agora é apenas acessar a seguinte URL: https://localhost:3000/

Tudo deve estar funcionando normalmente.
